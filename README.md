# Law Study Tool — 1L Edition

A Socratic AI study assistant for law school powered by Groq.

## Features
- 📖 Reading storage with course tags
- 🎓 Socratic tutor with cold-call + strict IRAC mode
- 📋 Auto case brief generator + rule polisher
- 🎯 Issue spotting drills with scored rubric
- 🃏 Flashcards with spaced repetition
- 📊 Progress tracking

## Setup

### Prerequisites
- Node.js 18+
- pnpm (`npm install -g pnpm`)
- Groq API key → https://console.groq.com

### Install
```bash
pnpm install
